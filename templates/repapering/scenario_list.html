Managing Document Requirements
How to Use This Section

This section allows you to define which documents are needed for different scenarios. You can add, edit, or remove document mappings as needed.

PDF Templates

For each requirement, you can attach or replace the PDF template. When you replace an old template with a new one, the old file is automatically removed from the system.

Tracking Changes

Every change you make—whether updating a mapping, replacing a template, or deleting a requirement—is securely logged. You can view the history of these actions if needed.

Import/Export

You can export your current scenario settings to a JSON file for backup purposes or import settings from a JSON file to bulk-load configurations between environments.
